Sample Plan
Beginner Plan (8 Weeks)

Weekly Schedule:
Monday: Rest or cross-train (swimming, cycling)
Tuesday: Run 20-30 minutes at a comfortable pace
Wednesday: Rest or light strength training
Thursday: Run 20-30 minutes, (1 min fast, 2 min slow)
Friday: Rest
Saturday: Long run (start with 30 minutes)
Sunday: Active recovery (walk, yoga)
Intermediate Plan (8 Weeks)

Weekly Schedule:
Monday: Rest or easy cross-train
Tuesday: Tempo run (20 min at a challenging pace)
Wednesday: Easy run (30-45 minutes)
Thursday: Speed work (400m repeats)
Friday: Rest
Saturday: Long run (start at 60 minutes)
Sunday: Active recovery
Advanced Plan (10 Weeks)

Weekly Schedule:
Monday: Rest or easy cross-train
Tuesday: Hill repeats (run uphill for a set time, jog down)
Wednesday: Easy run (45-60 minutes)
Thursday: Tempo run (30 min hard)
Friday: Rest or light strength training
Saturday: Long run (progressively increasing distance)
Sunday: Recovery run (30 minutes at an easy pace)
